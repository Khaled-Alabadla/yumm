# Contributing to Yumm Platform

Thank you for your interest in contributing to Yumm! We welcome everyone to help improve this Palestinian dining discovery platform.

## How to Contribute

### 1. Reporting Bugs
* Check the Issues tab to see if the bug is already reported.
* Open a new issue with a clear title and description of the bug.
* Include steps to reproduce the error and screenshots if available.

### 2. Suggesting Features
* Open an issue with the tag "enhancement".
* Explain the feature clearly and why it would be helpful for users or restaurant owners.

### 3. Submitting Code Changes
* Fork the repository and create a new feature branch.
* Write clean, documented code and test it locally first.
* Always update your `requirements.txt` file using `pip freeze` if you added new packages.
* Open a Pull Request with a detailed description of your changes.
